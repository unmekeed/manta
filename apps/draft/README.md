# Draft Engine

Рекомендации пиков драфта (Гл. 3, спринт 37). Контракт —
`proto/services.proto` → `DraftService.SimulateDraft(DraftState)`:
предсказанный винрейт Radiant текущего драфта + топ-5 пиков действующей
стороны (hero_id — Valve-ids из `libs/data/heroes.json`).

## Бейзлайн (engine/)

GNN из спеки требует 10^4+ матчей; здесь честные частоты со сглаживанием:
соло-винрейты, синергии пар в команде, направленные контрпики — из
PlayerMatchFeatures (составы есть только у реплей-матчей). Beta-prior к
0.5 (M_SOLO=40, M_PAIR=60) сознательно прижимает оценки редких пар к
нейтральным: на ~10^3 матчей модель не выдумывает уверенность. Итог —
аддитивная лог-оддс модель: sigmoid(Σ взвешенных вкладов), пустой драфт
даёт ровно 0.5, зеркальный драфт — зеркальную вероятность (тест).

## Запуск

```bash
make draft-serve       # gRPC :50053, метрики :9112
pytest tests/
```
