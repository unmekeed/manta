# AI Coach (Recommendation Engine)

План тренировки игрока из его отчётов + RAG-прецеденты (Гл. 3, Гл. 6.3,
спринт 38). Контракт — `RecommendationService.BuildPlan(player_id)` →
`TrainingPlan`; полный текст плана — в `resource_url` первого пункта с
префиксом `text:` (прото не меняем — источник истины).

## Двухслойный каркас

1. **Аналитический слой** (`engine/plan.py`, без LLM): детерминированные
   наблюдения из MatchReports — частота критических смертей и потерянный
   WP, доля смертей в рискованной позиции (Safety Index), слабый лейнинг,
   низкий импакт. Шаблонный план с цифрами и примерами матчей.
2. **LLM-слой** (`engine/llm.py`): опционально превращает те же
   наблюдения + прецеденты (Similarity.FindSimilar) в связный текст.
   `COACH_LLM_PROVIDER=anthropic COACH_LLM_API_KEY=...`; без ключа —
   мягкая деградация в шаблонный режим, ничего не блокируется.

## Запуск

```bash
make coach-serve       # gRPC :50054, метрики :9113
pytest tests/
```
